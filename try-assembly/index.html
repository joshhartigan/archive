Try Assembly

Assembly is a low-level programming language, used for interfacing almost directly with computer hardware - such a language can be described as "close to the metal".

Originally designed to be simpler than machine code and binary, assembly isn't used as much in modern times, thanks to higher level languages (e.g. C++, Python, etc.)

Assembly language varies with different specifications for different processors. This tutorial uses Intel's x86 assembly specification.

Firstly, let's do some arithmetic. Try ADD 8,9 to add two numbers together.
